# Data Analysis Project
## Simple Word Cloud Generator from Web Page

This project generates a word cloud based on the word count of a given web page. You simply provide the URL of the page, and the program will fetch the page content, process the text, count the frequency of each word, and use this data to generate a visual word cloud.

## Getting Started

### Prerequisites

You need Python 3.x and the following libraries:
- requests
- BeautifulSoup
- wordcloud
- matplotlib
